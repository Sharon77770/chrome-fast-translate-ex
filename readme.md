# 📑 Super PDF Translator

**Super PDF Translator**는 웹 서핑이나 PDF 문서 학습 시 흐름을 끊지 않고 즉시 번역을 도와주는 스마트 브라우저 확장 프로그램입니다. 



## ✨ 주요 기능

### 1. 🖱️ 지능형 스크롤 번역
- 웹페이지나 PDF에서 텍스트를 드래그하는 순간, 우측 사이드 패널에 즉시 번역 결과가 나타납니다.
- 학습 흐름을 방해하는 팝업 창 대신, 고정된 사이드바에서 편안하게 내용을 확인하세요.
- 사이트 정책에 따라 작동하지 않을 수 있습니다.

### 2. ⌨️ 강력한 단축키 지원
- 마우스 클릭 없이 키보드만으로 모든 기능을 제어할 수 있습니다.
- **패널 열기/닫기**: `Ctrl + Shift + Y` (Mac: `Cmd + Shift + Y`)
- **탭 전환**: `Ctrl + Shift + Right` (다음 탭으로 순환)

### 3. 📝 스마트 단어장 (캐싱)
- 번역했던 문장들이 자동으로 저장됩니다. 
- 별도로 기록할 필요 없이 '단어장' 탭에서 복습할 수 있으며, 데이터는 브라우저 로컬 저장소에 안전하게 보관됩니다.

### 4. 🔤 직접 입력 번역
- 드래그 번역 외에도 사용자가 직접 문장을 입력하여 번역할 수 있는 전용 탭을 제공합니다.

## 🚀 설치 방법 (개발자 모드)

1. 이 저장소를 `Clone` 하거나 `.zip` 파일을 다운로드하여 압축을 풉니다.
2. 크롬 브라우저에서 `chrome://extensions/` 주소로 이동합니다.
3. 우측 상단의 **[개발자 모드]**를 활성화합니다.
4. **[압축해제된 확장 프로그램을 로드합니다]**를 클릭하여 해당 폴더를 선택합니다.

## ⌨️ 단축키 안내

| 기능 | Windows/Linux | macOS |
| :--- | :--- | :--- |
| **사이드 패널 열기** | `Ctrl + Shift + Y` | `Cmd + Shift + Y` |
| **탭 전환 (순환)** | `Ctrl + Shift + →` | `Cmd + Shift + →` |
| **탭 바로가기** | `Alt + 1, 2, 3` | `Alt + 1, 2, 3` |

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3 (Modern UI), JavaScript (ES6+)
- **Storage**: Chrome Storage API (단어장 데이터 보관)
- **Library**: PDF.js (PDF 렌더링 지원)
- **API**: Google Translate Unofficial API (No API Key Required)
