# 📑 Super PDF Translator

**Super PDF Translator**는 웹브라우저와 PDF 내에서 드래그한 영어 텍스트를 즉시 한국어로 번역해주는 크롬 익스텐션입니다. 파파고 API의 유료화 및 제한을 대신하여, 별도의 API 키 없이도 사용할 수 있도록 경량화되었습니다.

## ✨ 주요 기능
- **PDF 번역 지원**: 내장된 PDF.js 뷰어를 통해 PDF 문서 내 텍스트를 드래그하여 바로 번역 가능.
- **사이드패널 UI**: 번역 결과가 브라우저 우측 사이드패널에 표시되어 읽기 흐름을 방해하지 않음.
- **No API Key**: 구글 번역 엔진(비공식)을 활용하여 복잡한 키 발급 과정 없이 설치 즉시 사용 가능.
- **경량화**: 불필요한 기능 없이 번역 본연의 기능에 집중.

## 🚀 설치 방법 (개발자 모드)
현재 이 프로젝트는 크롬 웹 스토어 등록 전 단계로, 아래 방법으로 설치할 수 있습니다.

1. 이 저장소를 `Clone` 하거나 `.zip` 파일로 다운로드 후 압축을 풉니다.
2. 크롬 브라우저에서 `chrome://extensions/` 주소로 이동합니다.
3. 우측 상단의 **[개발자 모드]** 스위치를 켭니다.
4. 좌측 상단의 **[압축해제된 확장 프로그램을 로드합니다]** 버튼을 클릭합니다.
5. 압축을 푼 폴더를 선택합니다.

## 🛠 사용 방법
1. 번역하고 싶은 웹페이지나 PDF 파일을 엽니다.
2. 번역할 문장을 마우스로 드래그합니다.
3. 우측 상단의 익스텐션 아이콘을 눌러 **사이드패널**을 엽니다.
4. 드래그한 텍스트가 자동으로 입력되며 번역 결과가 나타납니다.

## 📂 프로젝트 구조
- `manifest.json`: 익스텐션 설정 및 권한 관리
- `sidepanel.html/js`: 우측 번역창 화면 및 로직
- `background.js`: 텍스트 데이터 전달 및 백그라운드 프로세스
- `pdfjs/`: PDF 렌더링을 위한 오픈소스 라이브러리
